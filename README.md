# 🍅 Pomo Flow

> Aplicativo de produtividade baseado na técnica Pomodoro, construído com React."

## ✨ Features

- Temporizador Pomodoro customizável
- Pausas curtas e longas automáticas
- Histórico de sessões
- Músicas de fundo

## 🚀 Demo

https://pomo-flow.vercel.app/

## 🛠️ Tecnologias

- React
- Tailwind
- Firebase
- Javascript
- NodeJs

## 📦 Como rodar localmente

```bash
git clone https://github.com/AlanDev-br/Pomo-Flow
cd pomo-flow
npm install
npm run dev
```
